Entidad: Causa

Campos principales:
- id
- titulo
- descripcion
- fecha

Proposito:
Guardar las causas comunitarias que seran visibles en la plataforma para que los ciudadanos 
puedan conocerlas y apoyarlas.


Entidad: Apoyo

Campos principales:
- id
- causa_id
- usuario
- fecha

Proposito:
Registrar los apoyos que recibe cada causa para hacer seguimiento a la participacion ciudadana 
y mostrar cuantas personas respaldan cada iniciativa.


Entidad: Usuario

Campos principales:
- id
- nombre
- correo
- contrasena_cifrada
- fecha_registro

Proposito:
Gestionar el acceso a la plataforma, identificando a cada persona que interactua con el sistema 
y controlando sus permisos segun su rol.
